## Description

base project.

used Framework: Nestjs, Typeorm, Swagger

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start:local

# dev server mode
$ yarn run start:dev

# stage server mode
$ yarn run start:stage

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test:watch

# test coverage
$ yarn run test:cov
```
